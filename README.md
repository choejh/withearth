# 1. 기술 스택

* 웹 표준
  * HTML5
  * CSS3
  * JavaScript
  * jQuery
  * Bootstrap
* TOMCAT
* DBMS - MySQL
* 사용 언어 및 프레임워크
  * Java
  * Spring Framework
  * JSP, EL
  * Python
* 다양한 Api 사용
