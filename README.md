# 1. 기술 스택

<p align="center"><img width="664" alt="33" src="https://user-images.githubusercontent.com/71997976/111419783-6acc8b00-872d-11eb-916f-88df026135a2.PNG">

* 웹 표준
  * HTML5
  * CSS3
  * JavaScript
  * jQuery
  * Bootstrap
* TOMCAT
* DBMS - MySQL
* 사용 언어 및 프레임워크
  * Java
  * Spring Framework
  * JSP, EL
  * Python
* 다양한 Api 사용
  * jQuery, aJax, JSON 파싱
  * 카카오 맵 Api
  * 지오로케이션 Api
  * zxing api

* RESTful Api 구조
  * 서버 <--> 클라이언트 통신을 위한 REST 인터페이스 구현
* AWS 배포
  * EC2
  * RDS
